# NC County Analysis

The data in this repository consists of the county boundaries of North Carolina. The attributes include county name, DOT District number, and other related information. Data was provided by the [NC OneMap GeoPortal](http://data.nconemap.gov/geoportal/catalog/search/resource/details.page?uuid=%7BEDD7EAA4-F8B6-4B41-A5A4-77D3D85B16C6%7D) and downloaded from [NCDOT hosted on ArcGIS Online](http://ncdot.maps.arcgis.com/home/item.html?id=4d71f5b4c2474b1da1e43886e4d3184a).
